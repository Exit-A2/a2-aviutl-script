# a2-aviutl-script
 A2が作ったAviUtlのスクリプトやツールをまとめます
## ダウンロード・使い方
https://github.com/Exit-A2/a2-aviutl-script/releases/latest の「Source Code(zip)」をクリックしてください。<br>
スクリプトファイルの導入方法については[こちら](https://scrapbox.io/aviutl/スクリプトの導入方法)<br>
exoファイルは拡張編集のウィンドウにドラッグ&ドロップすることで使えます。<br>
exeファイルは実行ファイルなので、ダブルクリックすれば開けます。
## スクリプト一覧
### @A2.anm
#### 倍率クリッピング
- 倍率をかけてクリッピングできます

### @A2.obj
#### 角丸四角形
- モールスの副産物です

####  ランダム画像・数字から画像
- 鬼ほど使いにくいから使わないでください

### @すべあな.anm
#### 疑似カメラ制御
- 教育の奥に行く文字を再現する用です
- X・Y：出現時に画面が真っ白になってしまわないように微調整してください
- Z：カメラを0としたときのZ座標。これを直線運動させてください
- 基準距離：元の状態のZ座標を何とするか
- 実際に使用するときはZを中間点無視にして、出現時にシャープ、消滅時にぼかしを直線運動で、加えてモーションブラー・フェードもつけることをおすすめします

#### ランダム点滅
- 表裏の最後の字幕を再現します
- コマ落ち：時間制御のコマ落ちと同じ

### @すべあな.obj
#### モールス
- .のモールスです
- 時間：短音の場合0から1まで、長音の場合0から3まで
- 赤線：2番左モールスの赤線を追加

#### ...パルス
- ...の点字です
- 色1・2・3・4：パルスの色
- 実際に使用するときは3つのシーンに分け、一斉に並べてから分割していくことをおすすめします

#### ...バー(未完成)
- 未完成

#### 表裏リトマス
- 表裏の電飾です
- 時間：これを直線運動
- 色によって輪っかみたいになってしまったら、拡散の値を上げてください
#### アブジェ波紋
- アブジェの波紋です
- 半径を上げた時に発光しなくなってしまったら、環境設定から最大画像サイズを上げてください。画面の上下半分の対角線より大きくなるようにしてください
- かなり重いので、一度動画ファイルに書き出してからそれを加算モードで置いていくことをおすすめします
#### K2雨
- K²の雨です
- 量：画面上にある雨粒の量を指定します。上げ過ぎに注意
- 速度・風：px/sで指定します
- 拡散：°で指定します
- 速さ・風のドライバー感度：ドライバーにどれくらい従うか指定します。
- ドライバーをリセット：ドライバーを消したのに動かないとき、一瞬だけオンにしてください
- 性質上、プレビューだと実際よりゆっくりに見えることがあります。RAMプレビューなど使って速さは確認してください
#### K2雨ドライバー
- 複数のK2雨オブジェクトの「速さ」と「風」を一括で操作できます
- 必ずどのK2雨よりも上のレイヤーに配置してください
- 実際に使うときは、1つのシーンに全てのK2雨を配置し、レイヤーの表示/非表示で1つずつ出力してから加算モードで合成することをおすすめします
### VHS.exo
VHSエフェクトです。480x360用に作ってます。フレームレートを30FPSにするとなお良し
### AviUtlひらがな小さく.exe
AviUtl用に、ひらがなに自動で制御文字を入れてくれるアプリです。
※patch.aulを導入していないとテキストの表示がおかしくなる場合があります。
