# a2-aviutl-script
A2が作ったAviUtlのスクリプトやツールをまとめます
## ダウンロード・使い方
https://github.com/Exit-A2/a2-aviutl-script/releases/latest の「Source Code(zip)」をクリックしてください。<br>
スクリプトファイルの導入方法については[こちら](https://scrapbox.io/aviutl/スクリプトの導入方法)<br>
exoファイルは拡張編集のウィンドウにドラッグ&ドロップすることで使えます。<br>
exeファイルは実行ファイルなので、ダブルクリックすれば開けます。
## スクリプト一覧
### @A2.anm
#### 倍率クリッピング
- 倍率をかけてクリッピングできます

### @A2.obj
#### 角丸四角形
- モールスの副産物です

####  ランダム画像
- 鬼ほど使いにくいから使わないでください

#### 数字から画像
- 数字をトラックバーで指定して画像を表示できます。
- ファイルパスに%sを入れると、そこに数字が入ります。また、フォルダー階層には必ず/を使ってください。
- 例：C:/Users/exita2/song/%s.png

### @すべあな.anm
#### 疑似カメラ制御
- 教育の奥に行く文字を再現する用です
- X・Y：出現時に画面が真っ白になってしまわないように微調整してください
- Z：カメラを0としたときのZ座標。これを直線移動させてください
- 基準距離：元の状態のZ座標を何とするか
- 実際に使用するときはZを中間点無視にして、出現時にシャープ、消滅時にぼかしを直線移動で、さらにモーションブラー・フェードもつけることをおすすめします

#### ランダム点滅
- 表裏の最後の字幕を再現します
- コマ落ち：時間制御のコマ落ちと同じ

### @すべあな.obj
#### モールス
- .のモールスです
- 時間：短音の場合0から1まで、長音の場合0から3まで
- 赤線：2番左モールスの赤線を追加

#### ...パルス
- ...の点字です
- 色1・2・3・4：パルスの色
- 実際に使用するときは3つのシーンに分け、一斉に並べてから分割していくことをおすすめします

#### 表裏リトマス
- 表裏の電飾です
- 時間：これを直線移動
- 色によって輪っかみたいになってしまったら、拡散の値を上げてください

#### アブジェ波紋
- アブジェの波紋です
- 半径を上げた時に発光しなくなってしまったら、環境設定から最大画像サイズを上げてください。画面の上下半分の対角線より大きくなるようにしてください
- かなり重いので、一度動画ファイルに書き出してからそれを加算モードで置いていくことをおすすめします

#### K2雨
- K²の雨です
- 量：雨粒の量。上げ過ぎに注意
- 速度・風：px/sで指定
- 大きさ：雨粒の大きさ
- 風バリエーション：風の速さにバリエーションをつけます
- 奥行き：奥行きを指定します　上げると雨粒が見えにくくなるので注意
- 拡散：°で指定
- 前のオブジェクトから続ける：雨の動きを複数のイージングで制御したいときは、オブジェクトを分割し後の方のオブジェクトのこれを有効にします
- 性質上、プレビューが実際よりゆっくりに見えることがあります。RAMプレビューなど使って速さの確認を推奨します

### VHS.exo
VHSエフェクトです。480x360用に作ってます。フレームレートを30FPSにするとなお良し
### AviUtlひらがな小さく.exe
AviUtl・AviUtl2・YMM4用に、ひらがなに自動で制御文字を入れてくれるアプリです。
※patch.aulを導入していないとテキストの表示がおかしくなる場合があります。
